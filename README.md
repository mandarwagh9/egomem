# EgoMem

A research effort to build a **model-agnostic memory layer for robotics** — a
neutral `write` / `query` / `retrieve` layer that BOTH a world model AND a VLA
policy can call, fed by egocentric human task video (LeRobot v3 episodes, depth,
pose).

**Status:** early research. No validated result yet. This repo is driven by an
autonomous research-to-invention loop; **`PROGRESS.md` is the source of truth**
for where things stand and what happens next.

- `PROGRESS.md` — current phase, run log, next task.
- `research/` — lit map, gap statement, hypothesis, design.
- `experiments/` — one folder per run; raw logs + configs.
- `RESULTS.md` — append-only experiment results.
- `BIBLIOGRAPHY.md` — verified sources.
- `lib/`, `paper/` — populated once there is a defended result.

Full install/repro instructions land in Phase 7 (PACKAGE), not before.
